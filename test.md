# 마크다운(Markdown) 문법 샘플과 설명

마크다운은 텍스트에 간단한 기호를 붙여서 문서 서식을 지정하는 문법입니다. 아래는 자주 쓰이는 문법 예시와 설명입니다.

## 1. 제목 (Heading)

`#`의 개수로 제목의 크기(레벨)를 정합니다. `#`은 1개(가장 큼)부터 6개(가장 작음)까지 사용할 수 있습니다.

```markdown
# 제목 1
## 제목 2
### 제목 3
```

## 2. 강조 (Emphasis)

```markdown
*이탤릭체* 또는 _이탤릭체_
**굵게** 또는 __굵게__
***굵고 이탤릭체***
~~취소선~~
```

결과:

*이탤릭체*
**굵게**
***굵고 이탤릭체***
~~취소선~~

## 3. 목록 (List)

순서 없는 목록은 `-`, `*`, `+` 중 하나를 사용합니다.

```markdown
- 사과
- 바나나
  - 하위 항목
- 딸기
```

순서 있는 목록은 숫자와 마침표를 사용합니다.

```markdown
1. 첫 번째
2. 두 번째
3. 세 번째
```

## 4. 링크와 이미지

```markdown
[링크 텍스트](https://example.com)
![이미지 설명](https://example.com/image.png)
```

## 5. 코드 (Code)

인라인 코드는 백틱(`) 하나로 감싸고, 코드 블록은 백틱 3개(```)로 감쌉니다.

```markdown
인라인 코드 예시: `console.log("hi")`

```python
def hello():
    print("Hello, world!")
```
```

## 6. 인용문 (Blockquote)

```markdown
> 이것은 인용문입니다.
> 여러 줄도 가능합니다.
```

결과:

> 이것은 인용문입니다.
> 여러 줄도 가능합니다.

## 7. 표 (Table)

```markdown
| 이름 | 나이 | 직업 |
| ---- | ---- | ---- |
| 홍길동 | 30 | 개발자 |
| 김철수 | 25 | 디자이너 |
```

결과:

| 이름 | 나이 | 직업 |
| ---- | ---- | ---- |
| 홍길동 | 30 | 개발자 |
| 김철수 | 25 | 디자이너 |

## 8. 구분선 (Horizontal Rule)

`---`, `***`, `___` 중 하나를 한 줄에 단독으로 쓰면 구분선이 생깁니다.

## 9. 체크박스 (Task List)

```markdown
- [x] 완료한 작업
- [ ] 아직 안 한 작업
```

결과:

- [x] 완료한 작업
- [ ] 아직 안 한 작업
